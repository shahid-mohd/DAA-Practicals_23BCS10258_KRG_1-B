# Complete Controls Guide

## Mouse Controls

### Basic Drawing
- **Left Click + Drag**: Draw walls
- **Right Click + Drag**: Erase walls

### Point Selection
- **Ctrl + Left Click**: Set START point
- **Shift + Left Click**: Set END point

## Keyboard Shortcuts

### Algorithms
- `1` - Select BFS
- `2` - Select Dijkstra  
- `3` - Select A*

### Execution
- `SPACE` - Start/Pause
- `P` - Pause/Resume
- `R` - Reset
- `I` - Instant Run
- `C` - Clear Grid

### Maze Generation
- `G` - DFS Maze
- `D` - Recursive Division
- `B` - Binary Tree
- `O` - Random Obstacles

### File Operations
- `Ctrl+S` - Save Maze
- `Ctrl+L` - Load Maze
- `F12` - Screenshot

## UI Controls

### Grid Size
- Use sliders to adjust rows (10-60) and columns (10-80)
- Click "Apply Size" to resize grid
- Previous maze content is preserved when possible

### Speed Control
- Use slider to adjust visualization speed (1-100)
- Higher values = faster animation

### Algorithm Buttons
- Click to select algorithm
- Currently selected algorithm shows in status bar

### Execution Buttons
- **Start**: Begin algorithm visualization
- **Pause**: Pause during execution
- **Resume**: Continue paused execution
- **Reset**: Clear search states, keep maze

### Maze Generation Buttons
- Each button generates different maze type
- Automatically sets start (top-left) and end (bottom-right)
- **Clear Grid**: Remove all walls
